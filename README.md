# Cinquillo - Terminal Card Game (C) 🃏

A terminal-based implementation of the traditional Spanish card game "Cinquillo". This project was built to practice procedural programming, memory management, and game logic in C.

## 🚀 Features
* **Playable Terminal UI:** Uses ANSI escape codes for a colored, easy-to-read terminal interface.
* **Game Logic:** Full implementation of Cinquillo rules, including turn management, valid move validation, and win conditions.
* **CPU Players:** Play against automated opponents that calculate their own valid moves.
* **Native C:** Built using standard C libraries (`<stdlib.h>`, `<time.h>`, `<unistd.h>`) without relying on external dependencies.
